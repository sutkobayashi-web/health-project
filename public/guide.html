水槽と魚

アプリを開くと水槽が広がります。あなただけの魚を育てましょう。

1
卵が届く
最初に卵が届きます。名前をつけてあげてね。
2
ごはんを3回記録
食事の写真を3回撮ると、卵から透明な稚魚が生まれます。
3
種類が決まる
食事を続けると体に色がついて種類が判明！食事パターンで全30種類に進化します。
4
成長していく
稚魚 → 幼魚 → 若魚 → 成魚 → 大人 → 伝説の7段階。180日育てると大往生で褒賞！
⚠️ 放置するとどうなる？
3日 → 元気がなくなる
7日 → 病気になる
10日 → 死んでしまう（新しい卵はもらえます）
🔬 水質検査：月1回の検査で、見えない病気を早期発見できます。魚名タップ →「検査」から。
📖 おさかな図鑑：発見した魚はシークレットから解放！全30種コンプリートを目指そう。
🌊 仲間を見る：全社員の魚が一つの水槽に。元気ランキングTOP5も確認できます。
ごはんと水質のつながり

あなたが食べたものが、魚のごはんになります。食事内容が水質に直結！

🥗 野菜が多い → 水がきれいに、色が鮮やかに

🍟 揚げ物が多い → 水が濁る、魚が太る

🧂 塩分が多い → 魚がふらつく（高血圧リスク）

🍺 お酒が多い → 水が黄色くなる（肝臓リスク）

⚖️ バランス良好 → すべてが改善！キラキラに✨

投稿しない日は？ 自動で「フレーク」が与えられますが、魚はちょっと物足りなそう。あなたのごはんが一番嬉しいんだって🐟
歩数・血圧と魚

🏃 歩数記録（万歩計の数字を入力）

🚶 3,000歩 → 少し元気に

🏃 5,000歩 → 泳ぎが速くなる

🏃‍♂️ 7,000歩 → ウロコがキラキラ

🔥 10,000歩 → 全速力＋色が最高に鮮やか！

歩数は肥満・糖尿病リスクも軽減します。

🩺 血圧記録

マイページ設定でON → 血圧計の写真をAIが読み取り！
塩分の多い食事が続くと、魚の高血圧リスクも上がります。

魚に話しかけると：魚はあなたと同じ気持ちを語ります。「だるい...」「お腹重い...」— それはあなた自身の声かもしれません。
ごはんを投稿しよう

食事の写真を撮って投稿するだけ。みんなに見せよう！

1
メニュータブ →「今日のごはん見せて！」
カメラで食事を撮影。成分表示ラベルがあれば一緒に撮ると精度UP。
2
AIが分析
カロリー、栄養バランスを自動で分析してくれるよ。
3
みんなの反応
「食べたい！」「おいしそう！」「真似する！」のリアクションが届くよ。
連続投稿で木が育つ！ マイページで自分だけの木を育てよう。食事傾向で花の色が変わるよ。
ごはん番長： 今週一番「食べたい！」をもらった人が「ごはん番長」としてメニュー画面に表示されるよ！
話すだけで、声が届く
